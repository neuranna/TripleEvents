# Semantic reasoning takes place largely outside the language network

Code repository for the paper: 

Ivanova, A. A., Kauf, C., Gao, R., She, J. S., Kean, H. H., Goldhaber, T., Nieto-Castanon, A., Varley, R., Kanwisher\*\*, N. G., & Fedorenko\*\*, E. (2025). [Semantic reasoning takes place largely outside the language network](https://www.biorxiv.org/content/10.64898/2025.12.07.692873v1 ). *bioRxiv, 2025.12.07.692873*.

\*\* *co-senior* authors


**Contents:**
* Matlab scripts: GSS and fROI analysis code using the [spm_ss](https://github.com/alfnie/spm_ss) toolbox
* Participant_info: session IDs used for GSS and fROI analyses
* ! Stats_and_plots: R code for stats and plot generation, along with output plots and tables
* ! data: csv files with behavioral data, fROI responses, and fROI overlap results
